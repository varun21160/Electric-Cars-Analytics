# ⚡ Visualization Tool for Electric Vehicle Charge and Range Analysis

## 📌 Project Overview

Electric Vehicle Analytics is a data visualization project developed to analyze EV market trends, vehicle specifications, pricing, performance, efficiency, and charging infrastructure.

The project integrates multiple electric vehicle datasets with MySQL database management and Tableau visualization to convert raw data into meaningful insights. Interactive dashboards and stories are published using Tableau Public and integrated into a Flask web application for easy accessibility.

---

## 🎯 Purpose

The main objective of this project is to provide a centralized analytics platform that helps users understand electric vehicle data through interactive visualizations.

The platform helps analyze:

- EV charging infrastructure
- Electric vehicle brands and models
- Price comparison
- Vehicle performance
- Energy efficiency
- Body style categories
- Powertrain distribution
